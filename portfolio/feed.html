Fordpass Dealer Pay

 

About the project

Fordpas is a part of Ford, which deals with the customer payments to the dealers for their car maintenance and services, like oil change etc. This application was supposed to allow the dealers to manage and charge customers for the services.

Design thinking and taget which was set

The main idea was to make the work of the dealers easy while charging the customers and make the application user friendly with the modern visuals. The website was planned to be used on both computers and ipads, so we had to make sure that it is responsive.

Ford Motors, Room Scheduling

 

About the project

Ford is a global company and it is important to have all its employees all over the world who to be able to access and schedule room easily. This project is a global application where I was the designated Sr. Designer

Design thinking and taget which was set

Being an internal global application I had to make sure that the visual is comfortable for employees all over the world. It was important that I had to take in account how people from different countries think about design. There was a certain level of user research involved.
